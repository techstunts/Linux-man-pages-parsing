NAME

xvinfo - Print out X-Video extension adaptor information  

SYNOPSIS

xvinfo [-display displayname]  

DESCRIPTION

xvinfo prints out the capabilities of any video adaptors associated with the display that are accesible through the X-Video extension.  

OPTIONS

-display display
This argument allows you to specify the server to query; see X(7x).
 

ENVIRONMENT

DISPLAY
This variable may be used to specify the server to query.
