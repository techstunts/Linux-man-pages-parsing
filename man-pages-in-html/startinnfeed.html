STARTINNFEED

Section: Maintenance Commands (8)
Updated: Nov 7, 1997
 

NAME

startinnfeed - setuid root program to start innfeed  

SYNOPSIS

startinnfeed innfeed-options  

DESCRIPTION

Startinnfeed sets all resources (files opened, memory usage) to unlimited. It then executes innfeed(8) with the options specified.

James Brister
