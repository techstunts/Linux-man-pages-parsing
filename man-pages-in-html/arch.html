NAME

arch - print machine architecture  

SYNOPSIS

arch  

DESCRIPTION

arch is equivalent to uname -m.

On current Linux systems, arch prints things such as "i386", "i486", "i586", "alpha", "sparc", "arm", "m68k", "mips", "ppc".  

SEE ALSO

uname(1), uname(2)
