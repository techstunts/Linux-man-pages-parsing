NAME

printafm - Print the metrics from a Postscript font in AFM format using ghostscript  

SYNOPSIS

printafm fontname  

DESCRIPTION

This script invokes gs(1) to print the metrics from a font in AFM format. Output goes to stdout.  

SEE ALSO

gs(1)
