NAME

startfluxbox - customizable fluxbox start script  

SYNOPSIS

startfluxbox  

DESCRIPTION

Before starting fluxbox this script can set background images, styles, pixmaps, splash screens, keyboard layout and miscellaneous helper applications.  

FILES

~/.fluxbox/startup
If this file exists it is executed before starting fluxbox.
