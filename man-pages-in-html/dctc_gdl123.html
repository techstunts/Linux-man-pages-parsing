NAME

dctc - Direct Connect Text Client  

SYNOPSIS

dctc_gdl123 input_dir1 input_dir2 output_dir  

DESCRIPTION

dctc_gdl123 takes 2 corrupted GDLs (in input_dir1 and input_dir2) of the same file and build a new GDL (in output_dir) containing only parts identical in both corrupted GDLs. Beware, having identical bytes in both GDL does not mean bytes are valid.

 

AVAILABILITY

The latest version of hublist is inside DCTC package which can be obtained online from:

        http://ac2i.tzo.com/dctc/
