NAME

switchto - switch to a new virtual terminal (VT).  

SYNOPSIS

switchto vtnumber  

DESCRIPTION

switchto will change the current VT to the vt specified. Useful to change VTs when the hotkey sequence is unavailable for some reason, or from within scripts.
