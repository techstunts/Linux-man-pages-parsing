VOLNAME

Section: User Commands (1)
Updated: 5 May 2001
 

NAME

volname - return volume name  

SYNOPSIS

volname [<device-file>]

 

DESCRIPTION

Volname returns the volume name for a device formatted with an ISO-9660 file system, typically a CD-ROM.

The device file name can be specified on the command line. If omitted, it defaults to /dev/cdrom.
