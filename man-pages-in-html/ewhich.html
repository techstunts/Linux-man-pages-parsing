NAME

ewhich - Gentoo: program for resolving ebuild names

 

SYNOPSIS

ewhich [ -h | --help ] <package>

 

DESCRIPTION

ewhich is a little tool to find the actual ebuild file for a given package name. It can use simple package names (like xfree), package names with category (like x11-base/xfree) or DEPEND atoms (like >=x11-base/xfree-4.3.0).

 

BUGS
