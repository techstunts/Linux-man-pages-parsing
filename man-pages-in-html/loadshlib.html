NAME

loadshlib - Gnome Object Activation Directory  

SYNOPSIS

loadshlib

 

DESCRIPTION

loadshlib GOAD is the GNOME Object Activation Directory. It keeps track of the CORBA object implementations available on the system, and allows applications to activate these implementations or access currently running ones.  

OPTIONS

No options. This program is called internally by libgnorba27.
