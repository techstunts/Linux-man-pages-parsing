NAME

zap - a perl script used to kill processes  

USAGE

/usr/bin/mysql_zap [-signal] [-?Ift] pattern  

SYNOPSIS

zap [-I|-?] [-f] [-t]  

DESCRIPTION

zap
supports by executing
-I|-?
info
-f
force
-t
test
 

NOTE

If -f isn't given, ask user for confirmation for each process to kill. If signal isn't given, try first with signal 15 and after that with signal 9. If -t is given the processes is only shown on stdout.
