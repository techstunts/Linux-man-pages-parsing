NAME

pvf tools - utilities to convert various sound formats

 

DESCRIPTION

The pvf tools are a collection of tools to convert vgetty modem data to and from the 'raw modem data' format, and from that to and from various audio file formats (like .au or .wav). In addition, there are some tools to manipulate pvf files, like speed up files or cut off trailing noise.

A list of commands is below in the "see also" section.

You can run those commands with the -h switch for available options. Please also look at the individual contributed man pages.
