Section: User Commands (1)
 

NAME

jpilot - A palm pilot desktop for Linux/Unix  

SYNOPSIS

jpilot [-a] [-A] [-d] [-h] [-v]  

DESCRIPTION

J-Pilot is a desktop organizer application for the palm pilot and other Palm OS devices. It is similar in functionality to the one that 3Com/Palm distributes.  

OPTIONS

-v
displays version and exits.
-h
displays help and exits.
-d
displays debug info to stdout.
-a
ignore missed alarms since the last time this program was run.
-A
ignore all alarms, past and future.
-i
makes jpilot iconify itself upon launch.
 

ENVIRONMENT

The PILOTPORT, and PILOTRATE env variables are used to specify which port to sync on, and at what speed.

If PILOTPORT is not set then it defaults to /dev/pilot.
