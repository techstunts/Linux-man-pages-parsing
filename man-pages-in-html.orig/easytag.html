NAME

easytag - Tag modification utility  

SYNOPSIS

easytag [options]
easytag [directory]  

DESCRIPTION

easytag is an utility for viewing, editing and writing tags of your MP3, MP2, FLAC, Ogg Vorbis, MusePack and Monkey's Audio files. Its simple and nice GTK+ interface makes tagging easier.  

OPTIONS

--help
Display a short help
--version
Display version information
[directory]
Choose the directory to open at startup. It can be either an absolute path or a relative path.

FILES

EasyTAG will create a directory named .easytag in your home directory where it stores its config file and history files.
