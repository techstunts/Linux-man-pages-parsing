hublist

Section: User Commands (1)
Updated: March 2003
 

NAME

dctc - Direct Connect Text Client  

SYNOPSIS

hublist [options]  

DESCRIPTION

hublist retrieves list of hubs (servers) available for Direct Connect. This program is mainly designed to be used by programs not by users because the list of hubs is fairly hard to read.

hublist supports both uncompressed list, like the standard DC hub list, and bzip2 compressed list.  

OPTIONS

-h, --help
Display this help
--socks SocksProxyAddress SocksProxyPort [userID]
Set a SOCKS proxy to use. The SOCKS proxy port is mandatory. The userID to use on the SOCKS proxy is optionnal.
--proxy ProxyHost ProxyPort
Set a HTTP proxy to use.

 

Shell Variables

HUBLIST
if this variable is not defined, hublist will try to download the standard Direct Connect hub list. Using this variable, you also can define a set of alternate lists. Each list is described by an URL. If you have more than one list to define, use <tab> as separator. Note: if you want to use the standard DC hub list when the variable is defined, you must explicitly set its source URL.

 

AVAILABILITY

The latest version of hublist is inside DCTC package which can be obtained online from:
